# SquatLock Hub (ZTE Open)

Repositorio de la aplicación nativa para Firefox OS (ZTE Inari) que bloquea el flujo de trabajo para incentivar el ejercicio físico.

## 🚀 Instalación
1. Conectar el ZTE Open vía ADB.
2. Ejecutar el script de despliegue (o usar los comandos del manual en `docs/INSTALL.md`).

## 📁 Estructura
- `/src`: Código fuente Vanilla JS (ES5).
- `/build`: Paquete `application.zip` listo para sideload.
- `/docs`: Guía detallada de instalación y permisos de sistema.

## 🧬 Lógica de Bio-Hack
Timer de 45 minutos -> Bloqueo de UI -> 10 Sentadillas físicas (Acelerómetro) -> Persistencia en SD -> Desbloqueo.
